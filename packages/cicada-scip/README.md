# cicada-scip

SCIP-based language indexing for Cicada.

## Installation

```bash
pip install cicada-scip
```

## Supported Languages

- Python
- TypeScript / JavaScript
- Go
- Rust
- Java / Scala
- C / C++
- C# / VB
- Ruby
- Dart

## Usage

```python
from cicada_scip import SCIPReader, SCIPConverter
from cicada_scip.languages.python import PythonSCIPIndexer
```
